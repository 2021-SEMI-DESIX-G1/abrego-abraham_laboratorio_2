¿Qué es Javascript y en qué se asemeja a Java?
R= No tienen similitud, son totalmente diferentes.
1. JavaScript es un lenguaje interpretado, y Java es compilado.

2. Java se depura en dos fases y JavaScript en una: En Java primero verifica posibles errores de sintaxis durante la compilación y posteriormente
realiza la ejecución donde pueden surgir problemas de lógica. En Cambio JavaScript, solo se puede saber los errores en el momento de la ejecución,
a no ser que se usen herramientas especializadas como Visual Studio que interpretan el código en segundo plano.

3. Java es un lenguaje orientado a objetos puro, pero JavaScript está basado en prototipos.

4. Java es fuertemente tipado, y JavaScript es débilmente tipado: Las variables en Java solo puede ser de un tipo determinado y una vez definidas, estas no se pueden cambiar.
Por otro lado, en JavaScript las variables pueden cambiar de tipo de dato en cualquier momento, almacenado primero un texto, luego un número, luego una fecha, etc.

5. JavaScript es estándar, Java no.

6. JavaScript:
NO es lo mismo que Java.
NO es un subconjunto de Java.
NO es una versión de Java para programar páginas web.
NO comparte prácticamente nada con Java.
NO es más fácil que Java.
NO es solo un lenguaje para páginas web.

7. Si sabes programar en Java NO sabes programar en JavaScript. Y viceversa.
